# 專案目錄 📂

這裡是我為 Barry 和家人開發的所有工具專案。

## 🧰 核心工具

### assistant-toolkit
統一工具箱，整合所有專案功能。

### quick-answers
快速問答模組，關鍵字匹配自動回答。

### system-status
系統狀態監控，檢查服務和專案狀態。

## 📈 投資相關（Barry）

### investment-monitor ✅
投資組合監控系統。
- 股票報價（ONDS, TSLA）
- 組合績效追蹤
- 價格警報

### stablecoin-tracker ✅
穩定幣市場追蹤。
- USDT, USDC, DAI, USDD, TUSD, FDUSD
- 市值排名（總計 $262B）
- 脫鉤警報

### rwa-tracker ✅
RWA 市場追蹤。
- 代幣化資產市值（$19.4B）
- 國債、私募信貸、大宗商品

## ☀️ 生活相關

### morning-briefing ✅
每日早晨簡報。
- 天氣預報
- Google Calendar 行程
- 投資概覽
- Fintech 新聞
- 穩定幣/RWA 快訊

### family-reminders
家庭紀念日提醒（待補充日期）。

### story-time
Tim 的睡前故事生成器 📚
- 完整故事
- 簡短故事
- 互動故事開頭

## 📰 資訊相關

### smart-articles

### knowledge-intelligence

## 📈 金融市場

### forex-tracker 💱
匯率追蹤。
- USD/TWD, EUR/TWD, JPY/TWD, CNY/TWD
- 貨幣換算
- API: exchangerate-api.com

### crypto-tracker 🪙
加密貨幣追蹤。
- BTC, ETH, SOL, BNB, XRP, ADA, AVAX, DOT
- 價格和市值
- API: CoinGecko

### market-dashboard 📊
綜合市場儀表板。
- 整合所有金融追蹤器
- 完整市場報告
- Fintech 工作報告
