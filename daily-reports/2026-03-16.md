- 💌 感謝卡管理

**新增函數**:
```javascript
toolkit.getInvitationsBrief();            // 邀請卡快訊
toolkit.getInvitationsDashboard();        // 邀請儀表板
toolkit.getAllInvitationTemplates();      // 所有模板
toolkit.getInvitationTemplate('superhero'); // 特定模板
toolkit.selectInvitationTemplate('dinosaur'); // 選擇模板
toolkit.generateInvitationContent('小明'); // 生成邀請內容
toolkit.addInvitationGuest('小明', 'classmate', '0912345678'); // 新增賓客
toolkit.updateGuestRSVP('guest_id', 'confirmed', 2); // 更新 RSVP
toolkit.getPendingGuests();               // 待確認賓客
toolkit.getConfirmedGuests();             // 已確認賓客
toolkit.getEstimatedAttendees();          // 預估出席人數
toolkit.addThankYouCard('guest_id', '玩具車'); // 新增感謝卡
toolkit.getPendingThankYouCards();        // 待寄感謝卡
toolkit.getInvitationStats();             // 邀請統計
```

### 💰 派對預算追蹤

**新增檔案**: `family-center/birthday-budget.js`

**功能特點**:
- 💼 10 個預算類別（場地/裝飾/餐飲/蛋糕/娛樂/小禮/用品/邀請卡/攝影/雜項）
- 📊 建議預算分配（自動按比例分配）
- 💸 支出追蹤（含商家、日期、付款方式）
- 💰 收入追蹤（紅包等）
- ⚠️ 預算警告（80% 預警、超支警告）
- 📈 預算預測（根據平均支出預測）
- 📄 預算報告匯出

**新增函數**:
```javascript
toolkit.getPartyBudgetBriefNew();         // 預算快訊
toolkit.getPartyBudgetDashboard();        // 預算儀表板
toolkit.setPartyTotalBudget(10000);       // 設定總預算
toolkit.autoAllocateBudget();             // 自動分配預算
toolkit.addPartyExpense({ category: 'cake', description: '蛋糕訂購', amount: 1500 }); // 新增支出
toolkit.getCategoryBreakdown();           // 類別細分
toolkit.getBudgetWarnings();              // 預算警告
toolkit.getBudgetProjections();           // 預算預測
toolkit.addPartyIncome({ source: '紅包', amount: 500 }); // 新增收入
toolkit.getPartyNetCost();                // 淨支出
toolkit.exportBudgetReport();             // 匯出報告
```
