# RWA 市場追蹤器 🏦

追蹤代幣化真實世界資產 (RWA) 的市場規模和趨勢。

## 追蹤類別

| 類別 | 說明 |
|------|------|
| 國債 | 代幣化美國國債 (Treasuries) |
| 私募信貸 | 代幣化私募信貸 (Private Credit) |
| 大宗商品 | 代幣化黃金、原油等 |
| 房地產 | 代幣化房地產 |
| 股票 | 代幣化股票 |

## 主要協議

- Ondo Finance
- Centrifuge
- Maple Finance
- Goldfinch
- Backed Finance
- Matrixdock

## 功能

### 市場概覽
```bash
node src/index.js overview      # 市場總覽
node src/index.js treasuries    # 國債資料
```

### 協議排名
```bash
node src/index.js protocols     # 協議排名
```
