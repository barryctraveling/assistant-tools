# 🚀 快速參考卡

## 收藏文章

**跟我說：**
```
[貼上文章連結]
存到 Notion
```

**我會：**
1. 抓取文章 → 分析內容 → 存入 Notion → 更新知識庫

---

## 查詢知識庫

**問我：**
- 「我收藏了哪些 RWA 文章？」
- 「最近收藏了什麼？」
- 「知識庫有多少文章？」

## CLI 指令（進階）

```bash
# 查看知識庫統計
node assistant-helpers.js stats

# 搜尋文章
node assistant-helpers.js search "RWA"

# 生成週報
node weekly-report.js

# 主題報告
node topic-report.js RWA

# 執行測試
npm test
```
