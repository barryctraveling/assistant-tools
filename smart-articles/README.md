# 智慧文章收藏系統 Smart Article Collector

> 為 Barry 打造的個人知識管理工具

## 🎯 專案目標

不只是「存文章」，而是建立一個能夠：
1. **深度理解** - 智慧摘要、提取關鍵觀點
2. **自動整理** - 分類、標籤、關聯
3. **累積知識** - 建立可搜尋的知識庫
4. **產生洞見** - 發現文章之間的連結

## 📁 系統架構

```
smart-articles/
├── README.md           # 專案說明
├── collector.js        # 主程式：文章收集器
├── lib/
│   ├── fetcher.js      # 網頁內容抓取
│   ├── analyzer.js     # 內容分析與摘要
│   ├── categorizer.js  # 自動分類
│   ├── notion.js       # Notion API 整合
│   └── knowledge.js    # 知識庫管理
├── templates/
│   └── article.json    # Notion 頁面模板
├── data/
│   ├── knowledge-base.json  # 知識庫索引
│   └── tags.json            # 標籤系統
└── tests/
    └── test-article.js      # 測試腳本
```

## ✨ 核心功能

### 1. 智慧內容提取
- 自動識別文章主體（排除廣告、導航等）
- 提取標題、作者、發布日期
- 保留重要的引用和數據

### 2. 深度摘要
- 3-5 個關鍵重點（bullet points）
- 一段話總結（50字內）
- 重要引用提取（名人/專家說法）
- 關鍵數據/統計

### 3. 智慧分類
基於內容自動判斷：
- 🤖 AI 科技
- 🏦 金融科技/區塊鏈
- 🌍 全球趨勢
- 📱 社群熱門
- 📌 其他

### 4. 知識標籤
自動提取：
- 人物（誰說的、誰做的）
- 組織（公司、機構）
- 概念（技術、趨勢）
- 地點（國家、城市）

### 5. 關聯發現
- 與已收藏文章的主題關聯
- 相似觀點或對立觀點
- 時間軸上的發展脈絡
