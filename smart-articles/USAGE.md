# 智慧文章收藏系統 - 使用指南

> 給 AI 助理的操作手冊

## 🎯 系統目的

這個系統讓 Barry 可以透過簡單的指令，將任何文章收藏到 Notion，並自動：
1. 提取關鍵資訊
2. 生成摘要
3. 自動分類
4. 建立標籤
5. 關聯相似文章

## 📥 觸發方式

當 Barry 說以下任一指令時，觸發收藏流程：
- 「存到 Notion」
- 「收藏這篇」
- 「加到收藏」
- 「幫我存這篇文章」
- 直接貼 URL + 任何表示想收藏的意圖

## 🔄 處理流程

### 1. 抓取文章
```javascript
// 使用 web_fetch 工具抓取文章內容
const content = await web_fetch(url);
```

### 2. 分析內容
使用 `lib/analyzer.js` 分析文章：
- 自動判斷分類（5 大類）
- 提取關鍵字標籤
- 生成 50 字摘要
- 提取 3-5 個關鍵重點
- 找出重要引用
- 提取數據點

### 3. 查詢關聯
使用 `lib/knowledge.js` 查詢知識庫：
- 找出標籤相同的過去文章
- 建立知識連結

### 4. 寫入 Notion
使用 `lib/notion.js` 建立頁面：
- 結構化的頁面格式
- 包含所有分析結果
- 預留「我的筆記」空間

### 5. 更新知識庫
將新文章加入知識庫索引

## 📝 回覆格式

收藏完成後，回覆 Barry：

```
✅ 已收藏！

📰 [文章標題]
📁 分類：[分類]

📌 重點摘要：
• 重點一
• 重點二
• 重點三

🏷️ 標籤：#標籤1 #標籤2 #標籤3

🔗 相關文章：
- [過去收藏的相關文章標題]

👉 [Notion 連結]
```
