# 故事時間 📚

為 Tim（4歲9個月）設計的睡前故事生成器。

## 功能

### 完整故事
```bash
node index.js story
```
一個有開頭、中間、結尾的完整故事，包含道德啟示。

### 簡短故事
```bash
node index.js short
```
適合很累的時候，3-4 句話的小故事。

### 故事開頭
```bash
node index.js start
```
只有開頭，讓 Tim 或爸爸媽媽繼續編故事。

## 道德啟示

每個故事都會包含一個正面的教訓：
- 分享是最快樂的事
- 勇敢面對困難
- 朋友之間要互相幫助
- 說謊是不對的
- 要相信自己
- 每個人都有自己的特別之處
