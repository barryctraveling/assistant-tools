# 台灣金融科技追蹤器 🇹🇼

專門追蹤台灣的金融科技、銀行創新和監管動態。

為 Barry 在臺灣銀行創新實驗室的工作設計。

## 追蹤主題

| 主題 | 說明 |
|------|------|
| 🏛️ 金管會動態 | 政策、公告、法規 |
| 🏦 銀行創新 | 數位轉型、純網銀 |
| ₿ 加密貨幣監管 | VASP、虛擬資產 |
| 💴 數位新台幣 | 央行 CBDC 進展 |
| 💳 支付創新 | 行動支付、電子支付 |
| 🧪 金融沙盒 | 監理沙盒案例 |

## 使用

```bash
node src/index.js overview   # 概覽
node src/index.js work       # 工作報告
node src/index.js suggest    # 搜尋建議
```
